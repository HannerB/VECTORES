Ejercicio 2a — Componentes y módulo de un vector
Determine las componentes y el módulo de:
• Un vector en el plano cuyo origen es el punto $(1,2)$ y su extremo el punto $(-3,3)$.
• Un vector en $\mathbb{R}^3$ cuyo origen es el punto $(0,-1,2)$ y su extremo es el punto $(-3,1,3)$.

Solución

Vector $\vec{v}_1$ en $\mathbb{R}^2$:

Para obtener las componentes de un vector a partir de dos puntos, se restan las coordenadas del extremo menos las del origen: $\vec{v} = B - A$.

$$\vec{v}_1 = (-3, 3) - (1, 2)$$
$$\vec{v}_1 = (-3-1,\; 3-2)$$
$$\vec{v}_1 = (-4,\; 1)$$

El módulo (o norma) de un vector en $\mathbb{R}^2$ se calcula con la fórmula $\|\vec{v}\| = \sqrt{x^2 + y^2}$:

$$\|\vec{v}_1\| = \sqrt{(-4)^2 + 1^2} = \sqrt{16 + 1} = \sqrt{17}$$
Componentes: $\vec{v}_1 = (-4,\; 1)$   |   Módulo: $\|\vec{v}_1\| = \sqrt{17} \approx 4{,}1231$

Vector $\vec{v}_2$ en $\mathbb{R}^3$:

Se aplica el mismo procedimiento para tres dimensiones: $\vec{v} = B - A$.

$$\vec{v}_2 = (-3, 1, 3) - (0, -1, 2)$$
$$\vec{v}_2 = (-3-0,\; 1-(-1),\; 3-2)$$
$$\vec{v}_2 = (-3,\; 2,\; 1)$$

La norma en $\mathbb{R}^3$ se calcula con $\|\vec{v}\| = \sqrt{x^2 + y^2 + z^2}$:

$$\|\vec{v}_2\| = \sqrt{(-3)^2 + 2^2 + 1^2} = \sqrt{9 + 4 + 1} = \sqrt{14}$$
Componentes: $\vec{v}_2 = (-3,\; 2,\; 1)$   |   Módulo: $\|\vec{v}_2\| = \sqrt{14} \approx 3{,}7417$
Ejercicio 2b — Valor de $x$ para vectores perpendiculares
Encuentre el valor de $x$ para que los vectores sean perpendiculares:
• $A = (2, 3)$ y $B = (-1, x)$
• $C = (5, 3, 1)$ y $D = (2, x, 4)$

Solución

Dos vectores son perpendiculares (ortogonales) si y solo si su producto punto es igual a cero: $\vec{u} \perp \vec{v} \iff \vec{u} \cdot \vec{v} = 0$.

Par 1: $A = (2, 3)$ y $B = (-1, x)$

$$A \cdot B = (2)(-1) + (3)(x) = 0$$
$$-2 + 3x = 0$$
$$3x = 2$$
$$x = \frac{2}{3}$$
Verificación: $A \cdot B = (2)(-1) + (3)\!\left(\frac{2}{3}\right) = -2 + 2 = 0$   ✓
$x = \dfrac{2}{3}$

Par 2: $C = (5, 3, 1)$ y $D = (2, x, 4)$

$$C \cdot D = (5)(2) + (3)(x) + (1)(4) = 0$$
$$10 + 3x + 4 = 0$$
$$14 + 3x = 0$$
$$3x = -14$$
$$x = -\frac{14}{3}$$
Verificación: $C \cdot D = (5)(2) + (3)\!\left(-\frac{14}{3}\right) + (1)(4) = 10 - 14 + 4 = 0$   ✓
$x = -\dfrac{14}{3}$
Ejercicio 2c — Operaciones con vectores $\vec{u}$ y $\vec{v}$
Dados los vectores $\vec{u} = -\hat{\imath} + 2\hat{\jmath} + 3\hat{k}$ y $\vec{v} = 3\hat{\imath} + \hat{\jmath} - 2\hat{k}$, encuentre: la norma de cada vector, el producto punto $\vec{u} \cdot \vec{v}$ y $\vec{v} \cdot \vec{u}$, el producto vectorial $\vec{u} \times \vec{v}$ y $\vec{v} \times \vec{u}$, la distancia entre ellos y el ángulo entre los vectores.

Solución

Los vectores en forma de componentes son: $\vec{u} = (-1, 2, 3)$ y $\vec{v} = (3, 1, -2)$.

1. Norma de cada vector

La norma de un vector en $\mathbb{R}^3$ se define como $\|\vec{w}\| = \sqrt{w_1^2 + w_2^2 + w_3^2}$.

$$\|\vec{u}\| = \sqrt{(-1)^2 + 2^2 + 3^2} = \sqrt{1 + 4 + 9} = \sqrt{14}$$
$$\|\vec{v}\| = \sqrt{3^2 + 1^2 + (-2)^2} = \sqrt{9 + 1 + 4} = \sqrt{14}$$
$\|\vec{u}\| = \sqrt{14} \approx 3{,}7417$   |   $\|\vec{v}\| = \sqrt{14} \approx 3{,}7417$

2. Producto punto

El producto punto (o producto escalar) se define como $\vec{u} \cdot \vec{v} = u_1 v_1 + u_2 v_2 + u_3 v_3$. Es una operación conmutativa, es decir, $\vec{u} \cdot \vec{v} = \vec{v} \cdot \vec{u}$.

$$\vec{u} \cdot \vec{v} = (-1)(3) + (2)(1) + (3)(-2) = -3 + 2 - 6 = -7$$
$$\vec{v} \cdot \vec{u} = (3)(-1) + (1)(2) + (-2)(3) = -3 + 2 - 6 = -7$$
$\vec{u} \cdot \vec{v} = \vec{v} \cdot \vec{u} = -7$

Se confirma la propiedad conmutativa del producto punto.

3. Producto vectorial

El producto vectorial se calcula mediante el determinante de una matriz $3 \times 3$ formada por los vectores unitarios y las componentes de ambos vectores. A diferencia del producto punto, el producto vectorial es anticonmutativo: $\vec{v} \times \vec{u} = -(\vec{u} \times \vec{v})$.

$$\vec{u} \times \vec{v} = \begin{vmatrix} \hat{\imath} & \hat{\jmath} & \hat{k} \\ -1 & 2 & 3 \\ 3 & 1 & -2 \end{vmatrix}$$

Expandiendo por cofactores a lo largo de la primera fila:

$$\hat{\imath}\big[(2)(-2) - (3)(1)\big] - \hat{\jmath}\big[(-1)(-2) - (3)(3)\big] + \hat{k}\big[(-1)(1) - (2)(3)\big]$$
$$= \hat{\imath}(-4 - 3) - \hat{\jmath}(2 - 9) + \hat{k}(-1 - 6)$$
$$= -7\hat{\imath} + 7\hat{\jmath} - 7\hat{k}$$
$\vec{u} \times \vec{v} = (-7,\; 7,\; -7)$

Por la propiedad anticonmutativa:

$$\vec{v} \times \vec{u} = -(\vec{u} \times \vec{v}) = -(-7, 7, -7) = (7, -7, 7)$$
$\vec{v} \times \vec{u} = (7,\; -7,\; 7)$

4. Distancia entre los vectores

La distancia entre dos vectores se calcula como la norma de su diferencia: $d(\vec{u}, \vec{v}) = \|\vec{u} - \vec{v}\|$.

$$\vec{u} - \vec{v} = (-1-3,\; 2-1,\; 3-(-2)) = (-4,\; 1,\; 5)$$
$$\|\vec{u} - \vec{v}\| = \sqrt{(-4)^2 + 1^2 + 5^2} = \sqrt{16 + 1 + 25} = \sqrt{42}$$
$d(\vec{u}, \vec{v}) = \sqrt{42} \approx 6{,}4807$

5. Ángulo entre los vectores

El ángulo $\theta$ entre dos vectores se obtiene a partir de la relación entre el producto punto y las normas: $\cos\theta = \dfrac{\vec{u} \cdot \vec{v}}{\|\vec{u}\| \cdot \|\vec{v}\|}$.

$$\cos\theta = \frac{\vec{u} \cdot \vec{v}}{\|\vec{u}\| \cdot \|\vec{v}\|} = \frac{-7}{\sqrt{14} \cdot \sqrt{14}} = \frac{-7}{14} = -\frac{1}{2}$$
$$\theta = \arccos\!\left(-\frac{1}{2}\right) = 120°$$
$\theta = 120°$

El ángulo es obtuso ($> 90°$), lo cual es consistente con el hecho de que el producto punto es negativo ($\vec{u} \cdot \vec{v} = -7 < 0$).

Ejercicio 2d — Área de un paralelogramo
Calcule el área de un paralelogramo cuyos lados son los vectores $(3, 2, 1)$ y $(-1, 5, 7)$.

Solución

El área de un paralelogramo formado por dos vectores $\vec{a}$ y $\vec{b}$ es igual a la magnitud de su producto vectorial: $\text{Área} = \|\vec{a} \times \vec{b}\|$.

Paso 1: Calcular $\vec{a} \times \vec{b}$

Con $\vec{a} = (3, 2, 1)$ y $\vec{b} = (-1, 5, 7)$:

$$\vec{a} \times \vec{b} = \begin{vmatrix} \hat{\imath} & \hat{\jmath} & \hat{k} \\ 3 & 2 & 1 \\ -1 & 5 & 7 \end{vmatrix}$$

Expandiendo por cofactores:

$$\hat{\imath}\big[(2)(7) - (1)(5)\big] - \hat{\jmath}\big[(3)(7) - (1)(-1)\big] + \hat{k}\big[(3)(5) - (2)(-1)\big]$$
$$= \hat{\imath}(14 - 5) - \hat{\jmath}(21 + 1) + \hat{k}(15 + 2)$$
$$= 9\hat{\imath} - 22\hat{\jmath} + 17\hat{k}$$
$\vec{a} \times \vec{b} = (9,\; -22,\; 17)$

Paso 2: Calcular la magnitud

$$\|\vec{a} \times \vec{b}\| = \sqrt{9^2 + (-22)^2 + 17^2}$$
$$= \sqrt{81 + 484 + 289}$$
$$= \sqrt{854}$$
$\text{Área} = \sqrt{854} \approx 29{,}2233$ unidades cuadradas
